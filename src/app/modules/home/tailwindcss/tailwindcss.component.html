Tailwind CSS A set of classes already available?

For those unfamiliar with Tailwind CSS, it is a highly customisable low-level CSS framework with helper classes for almost every CSS rule. Basically, an Angular project is composed of a set of modules containing components. Each component has a TypeScript file (.ts), an HTML file (.html) and a CSS, SCSS or SASS file (.css, .scss, .sass) depending on the design choice. With Tailwind, you can save a lot of effort by not having to create your own CSS classes as they are already present in most cases directly in Tailwind.
